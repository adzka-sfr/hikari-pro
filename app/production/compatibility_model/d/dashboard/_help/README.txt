Beberapa rule dalam pembuatan petunjuk kerja :
1. Nama file harus berupa "guide"
2. File menggunakan format pdf (.pdf)

Panduan dalam penulisan petunjuk kerja :
3. Direkomendasikan membuat petunjuk kerja menggunakan Word untuk kemudian di export ke format pdf
4. Petunjuk kerja harus memiliki daftar isi
5. Petunjuk kerja harus memiliki nomor halaman
6. Petunjuk kerja harus memiliki footer yang menginformasikan nama sistem
7. Layout menggunakan ukuran normal :
    a. Top : 2,54 cm
    b. Left : 2,54 cm
    c. Right : 2,54 cm
    d. Bottom : 2,54 cm
8. Font menggunakan Calibri (body)
9. Ukuran font
    a. Judul : 48
    b. Isi : 12
